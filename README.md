- Program menggunakan Scanner untuk membaca input dari pengguna melalui konsol.
- Pengguna diminta untuk memasukkan sebuah bilangan bulat.
- Program memverifikasi apakah input tersebut benar-benar bilangan bulat.
- Jika valid, program akan:
- Mengecek apakah bilangan tersebut habis dibagi 2 (n % 2 == 0)
- Menampilkan hasil apakah bilangan tersebut genap atau ganjil
- Jika input bukan bilangan bulat, program akan menampilkan pesan kesalahan dan keluar.
